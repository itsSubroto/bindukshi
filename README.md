## It is a jewellery website
- Fronend : HTML, CSS, JS
- Backend : PHP
- Database: MySQL

## User Features
- User can sign in
- Login
- Add product to wishlist
- Add product to cart
- Proceed to order

## Admin Features
- Admin can add product
- Define categories
- Handle customers/ users
- Can add multiple admin
- It has a dashboard for easy to handle
